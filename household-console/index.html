Home console

TV view is large-type and rotates when idle; Manage is the household editor (add to Home Screen on iPad for a PWA). Optional Supabase sync shares one household across Silk, iPad, and laptops.

TV dashboard Manage (PWA)

Fire TV: open /tv/ in Silk or wrap with Amazon Web App Tester. Sync: from repo root run supabase link then supabase db push, or paste supabase/migrations/*_household_sync.sql in the SQL editor. Deploy shared/sync-config.json, then use the same sync key on each device (see Manage).
